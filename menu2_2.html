파일 월별 정리하기

질문
다음에 제시된 조건에 따라 파이썬 코드를 생성해줘 # 조건 1. 현재 작업 폴더 하위의 'scandata' 폴더에 있는 파일들을 월별로 정리할거야 2. 날짜 정보는 파일이름으로 판별할 수 있어. 파일이름은 "년_월_일"로 구성되어 있어. 3. scandata 폴더에 있는 파일들을 현재 작업폴더 하위에 organized 폴더를 생성한 후 월별로 폴더를 생성해서 분류해줘
답변
import os import shutil # 현재 작업 폴더 하위에 "organized" 폴더 생성 organized_folder = "organized" os.makedirs(organized_folder, exist_ok=True) # "scandata" 폴더의 파일들을 가져오기 scandata_folder = "scandata" files = os.listdir(scandata_folder) # 파일들을 월별로 정리 for file_name in files: # 파일 이름에서 년도, 월 추출 year, month = map(int, file_name.split('_')[:2]) # 월별 폴더 경로 생성 month_folder = os.path.join(organized_folder, f"{year}_{month:02d}") # 월별 폴더가 없으면 생성 os.makedirs(month_folder, exist_ok=True) # 파일을 월별 폴더로 이동 source_path = os.path.join(scandata_folder, file_name) destination_path = os.path.join(month_folder, file_name) shutil.move(source_path, destination_path) print("파일 정리가 완료되었습니다.")
